1. Die ZIP-Datei in der App „Dateien“ sichern.

2. Die ZIP-Datei entpacken (unter iOS z. B. mit der App „iZip“).

3. Die Datei „index.html“ aus dem entpackten Ordner öffnen und über die Funktion „Offline speichern“ in „Dateien“ oder in einer Musik-App sichern. 

4. Die Anwendung kann anschließend vollständig offline genutzt werden und speichert alle Daten lokal auf dem Gerät.
_____________________
1. Save the ZIP file in the Files app.

2. Unzip the ZIP file (on iOS you can use an app like “iZip”).

3. Open the file “index.html” from the extracted folder and use the “Save offline” function to store it in Files or in a music app.

4. The application can then be used completely offline and stores all data locally on the device.
